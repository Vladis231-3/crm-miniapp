ОДНИМ КЛИКОМ
=============

Если exe уже собран (лежит в этой папке):
   запустите  start.bat

Если exe не собран или нужна пересборка:
   запустите  build-and-run.bat
   (автоматически установит всё и соберёт)


ЧТО ПРОИСХОДИТ
===============
Двойной клик на start.bat (или atmosfera-backend.exe):
   1. Запускает сервер на http://127.0.0.1:8000
   2. Открывает браузер с CRM
   3. Автоматический вход под админом (ID 473645426)
   4. База данных — Supabase PostgreSQL (общая с ботом)


НАСТРОЙКА
=========
Отредактируйте файл .env рядом с exe.
Сейчас стоит ваша Supabase БД.


ТРЕБОВАНИЯ ДЛЯ ПЕРЕСБОРКИ
==========================
- Python 3.10–3.12
- Node.js 18+
